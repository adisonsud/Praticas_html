ID y CLASS

El ID nos sirve para poder acceder al elemento del DOM de una forma mas facil, se puede usar con CSS pero no es recomendable, utilizar id para enlazar con javascript, jquery, php. El id no se puede repetir en la pagina

Class nos sirve para darle estilos con CSS a uno o varios elementos. Class si se puede repetir en la pagina
